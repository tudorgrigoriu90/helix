# Audio & Asset Licenses

Per TDD §21 Q3, music and SFX use royalty-free libraries (Epidemic Sound, Artlist, Freesound, Sonniss bundles) plus 3–5 custom signature stings ($500–$1,000 from a contractor).

This folder holds a license file per track and SFX used. Required at App Store / Play Store submission and surfaced in the in-game S096 About + Licenses screen.

Conventions:

- One file per source (e.g. `epidemic-sound-2026.md`, `freesound-floor-tiles.md`).
- Include: source URL, license version, dates active, scope (which audio assets it covers), and renewal terms.
